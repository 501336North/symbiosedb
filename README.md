# SymbioseDB

> The first DAO-ready database. Web3 + AI in one unified platform.

## What is SymbioseDB?

SymbioseDB is a unified multi-database platform that combines four powerful database types in one API:

- **PostgreSQL** - Battle-tested relational database with ACID transactions
- **Vector Search** - Semantic search with pgvector for AI applications
- **Graph Database** - Apache AGE for complex relationship queries
- **Blockchain** - Ethereum L2 attestations for immutable audit trails

## Features

### 4 Databases, 1 API
- Unified query interface across all database types
- Intelligent query routing based on patterns
- SAGA pattern for distributed transactions

### DAO & Web3 Native
- Built-in blockchain attestations on 16+ networks
- Immutable audit trails for compliance
- Multi-DAO support with isolated data
- Cryptographic proof of data integrity

### AI-Powered
- RAG (Retrieval-Augmented Generation) pipelines
- Multi-query fusion for better search results
- Citation verification to prevent hallucinations
- Document processing and embedding management

### Production-Ready
- Authentication (JWT + API Keys)
- Monitoring (Prometheus + Winston)
- Rate limiting & security hardening
- Real-time subscriptions via WebSocket
- 821 tests passing ✅

## Quick Start

```bash
# Create a new app
npx create-symbiosedb-app my-app

# Start developing
cd my-app
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see your app running!
